### File Naming Convention
Use descriptive, lowercase names with underscores:
- âś… `sensor_mounting_closeup.jpg`
- âś… `complete_assembly_front_view.png`

### Image Requirements
- **Format:** JPG for photos, PNG for diagrams/screenshots
- **Resolution:** Minimum 1280x720 for main images
- **File Size:** Keep under 2MB (compress if needed)
- **Quality:** Clear, well-lit, in focus
- **Background:** Clean, uncluttered

### Recommended Tools
- **Photo Editing:** GIMP, Photoshop, or online tools
- **Compression:** TinyPNG, ImageOptim
- **Annotation:** Draw.io, Inkscape, or Paint.NET
- **Screenshots:** Built-in OS tools or Lightshot

## đź“¸ Photography Tips

### For Assembly Photos
- Use good lighting (natural light or bright LED)
- Show multiple angles
- Include ruler/measurement reference
- Highlight important connection points
- Use macro mode for closeups

### For Demo Photos
- Ensure safety (never photograph while actively driving)
- Show the device in use (parked car)
- Capture serial monitor output
- Demonstrate proper wearing position

### For Circuit Diagrams
- Use contrasting colors
- Label all components
- Show pin numbers clearly
- Include legend/key
- Export in high resolution
